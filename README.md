## 基于SpringBoot的简易网上商城。

### 用途

- Java课程设计
- 毕业设计（需二次开发）

### 特点/缺陷

 - 模拟面部识别支付（Js调用摄像头，无实际实现）
 - 收货地址与订单无关联（购买时用户没有选择收货地址）
 - 购物车跳转支付有问题
 - 无分页
 - 搜索结果无法点击跳转商品详情
 - 个别页面css样式问题

### 功能模块

 - 用户登录注册
 - 用户信息管理（查看用户信息，修改电话）
 - 收货地址管理
 - 购买支付商品
 - 用户订单管理（用户查询历史订单，后台处理订单）
 - 后台商品管理（增删改查商品，修改商品数量）

### 实体
 - 用户
 - 商品
 - 商品类型
 - 订单
 - 订单详情
 - 收货地址
